# basecamp-questions
Questions for checking student understanding of Basecamp curriculum topics.
Within each topic, the questions are roughly ordered from easy to hard. Feel free to make a PR if you want to share.

### Topics

#### Introduction
#### Staying Motivated and Swallowing the Elephant
#### Computational Thinking vs. Syntax
* How is computational thinking different from regular thinking?
* What is syntax? (What does the word "syntax" mean?)
#### HTML & CSS Basics
* Go to [codepen.io](https://codepen.io/) and have the student create a `<div>` with some text in it. Then, use a CSS class to give it some styling (e.g., make it white text on a blue background, etc.) 
#### Variables & Data Types
* Make a string, array, object, and boolean variable.
#### Operators
* How many different operators can you name? (Have student describe what each ones does. Then discuss any ones they missed.)
#### Objects
* How do you make an object in JavaScript?
* What is an object property? 
* How do you access property values from outside of the object?
